# Snoop Parlante

Applicazione Android interattiva che simula un snoop parlante con animazioni Lottie.

## Funzionalità

-  Ripete la voce dell'utente a velocità 2x
-  Riproduce canzoni in sequenza con animazioni dinamiche
-  Emette versi casuali
-  Controllo torcia manuale e automatico
-  Sensore di luminosità con effetti visivi
-  Modalità muto

## Requisiti

- Android 13 (API 33) o superiore
- Android Studio Arctic Fox o successivo

## File Audio

I file audio vanno inseriti in `app/src/main/res/raw/`:
- `verse1.mp3`, `verse2.mp3`, `verse3.mp3` - Versi del pappagallo
- `song1.mp3`, `song2.mp3`, `song3.mp3` - Canzoni
- `scared.mp3` - Suono di paura

## Installazione

1. Clona il repository
2. Apri il progetto in Android Studio
3. Aggiungi i file audio in `res/raw/`
4. Sync Gradle
5. Run su dispositivo Android
